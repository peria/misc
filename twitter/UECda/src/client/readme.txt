
 クライアント ver 0.21

1.コンパイルと使い方について
 
 まず client/ に移動し、コンパイルしてください。次にサーバを立ち上げた
 状態で、クライアントを5つ実行するとゲームが始まります。
 
 詳細は、install.txtを参照してください。

2.再配布に関して
 
 改変したものを配布するのは自由ですが、その場合、オリジナルのパッケー
 ジも付属してください。また、改変したものがオリジナルではないことがわ
 かるようにしてください。
