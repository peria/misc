# 大貧民AIの開発キットをC++化してみたい

# メモ

Linux (ubuntu) で `server/` ディレクトリをビルドするのには

```
$ sudo apt-get install libxpm-dev
$ sudo apt-get install libxaw7-dev
```

っぽいことが必要。1つ目のライブラリは `configure` で確認されるが、2つ目は確認されない。
