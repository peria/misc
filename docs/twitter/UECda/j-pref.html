9. クライアント作成環境の構築(Java版)

標準クライアントやサーバーソフトウェアを使用するためには、 それぞれの言語が使用できる環境が必要になります。 この節では、Javaの環境を構築する方法を説明します。

9.1 動作環境

本プログラムはJVMバージョン1.5以上で動作します。 バージョンの確認方法はコマンドプロンプトで以下のコマンドを実行してください。 

java -version

ただし、大会ではバージョン1.5を使用しますので、プログラムのコンパイルは 1.5を使用してください。

9.2 開発キットのダウンロード

UECコンピュータ大貧民大会のWeb サイト[uecda.nishino-lab.jp] にアクセスし、 「ダウンロードページ」のリンクをクリックしてください。 開発キットの最新版へのリンクをクリックすると、 クライアント・プログラムの開発や動作確認に必要なファイル一式がダウンロードできます。

開発キットはjar形式で圧縮されています。 jarファイルは解凍して使っていただいても構いませんが、 本マニュアルでは解凍しないで使う方法について説明します。

まずは作業フォルダを作成します。ここでは /workspace/daihinmin/ で作業すると仮定します。 /workspace/daihinmin/フォルダの中にダウンロードしたdaihinmin.jarを移動してください。 これで準備は完了です。
